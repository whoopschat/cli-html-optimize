# cli-html-optimize
> html optimize cli

### install
```
> npm install cli-html-optimize --save-dev
```

### usage
```
> cli-html-optimize
Usage: cli-html-optimize [options]
  --src                src path
  --resMerge           style and js file merge
  --resMd5             create res .md5 file
  --zipFileName        zip file name
  --zipMd5File         zip md5 file name
  --help               show help
```
